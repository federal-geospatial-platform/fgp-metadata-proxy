AWS_TRANSLATE

Documentation du Custom Transformer FME

Description

This custom transformers translates FME attributes and/or lists in either English or French using AWS translation services. The attribute names or list to translate are specified using FME Published Parameters

Note1: Before translating an attribute the CT will verify if the destination attribute name exists. When the name exists, if the destination attribute value is different from the source attribute value to translate, the CT considers that the translation is done and the process stops for this attribute. If destination attribute value is the same, the CT will try to guess (using AWS translation service) the language of the source attribute value to translate and adjust the translation accordignly.

Note2:: When an attribute and or list is translated, the CT always creates 2 new attributes with the suffix fr and en. It keeps the original attribute name as is.

Input Ports

  • INPUT: The features to translate.

Output Ports

  • OUTPUT: The features with translated attributes and/or lists
  • TRANSLATION_PROBLEM: The features with a translation problem.

Parameters

  • ACTIVATE_TRANSLATION: Flag to enable or disable the translation. Ca take 2 values: Yes: Translate the attributes; No: Do not translate the attributes. If the flag is set to No, the transformer will copy the text to translate in the output attribute and add the prefix: Translation of.
  • MODE: Flag for the direction of the translation. can take 2 values: English-->French for an English to French translation; Franch-->English for a French to English translation;
  • ATTR_2_TRANSLATE: Coma separated list of FME attribute to translate;
  • ATTR_LIST_2_TRANSLATE: Coma separated list of FME lists to translate.

AWS_TRANSLATE.process_feature(feature)[source]

This method translate string using AWS translate tools.

The FME attributes to translate are stored in the FME attribute: _attr2translate The FME list to translate are stored in the FME attribute: _attrlist2translate. The translation mode in contained in the attribute: _mode. If mode equals: English–>French An english to Frech is done if the value equals: French–>English a French to English translation is done. The FME attribute _aws_activation contains the translation flag. If set to Yes the translation is done; if set to No the translation is not done.

AWS_TRANSLATE.set_en_fr_attributes(feature, att_name_to_translate, text_to_translate, translation_mode)[source]

Set the attribute suffix of the attribute to translate depending on the translation direction.

If the translation direction is en ==> fr, the suffix “_en” is added to the attribute to translate. If the translation direction is fr ==> en, the suffix “_fr” is added to the attribute to translate. If the attribute (including the suffix) to create exists and their attribute values are different then it’s problematic and the attribute value with the suffix is replaced by the original attribute name to translate.

Parameters

  • feature (FmeFeature object) – Feature object to process

  • att_name_to_translate (String) – Name of the attribute to translate

  • text_to_translate (String) – Text string to translate

  • translation_mode (String) – Flag to indicate the type of translation. Can take two values: English–>French for an English to French tranlation; French–>English for a French to English translation
