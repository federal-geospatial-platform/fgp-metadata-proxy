GEOSPATIAL_DATA_VALIDATOR

Documentation du Custom Transformer FME

Description

This custom transformer fan out the metadata records between three types of metadata (GeoSpatial, Non GeoSpatial and Unknown). In order to determine the type of metadta record, the transformer is using two strategies: firstly by checking if the format has been published in the FGP (if positive it is a GeoSpatial record); secondly by checking the content of a list of keywords againts the content of an attribute in the metadata record (if there is a match it is a GeoSpatial record)

Input Ports

  • LOOKUP_TABLE_INPUT: Input information from the lookup table

  • DATA_INPUT: The input metadata records to process.

Output Ports

  • GEOSPATIAL: Output port for all gepspatial metadata record

  • NON-GEOSPATIAL: Output port for all non geospatial metadata record

  • UNKNOWN_FORMAT: Output port for metadata record that cannot be mapped to geospatial or non geospatial

Parameters

  • KEYWORD_SEARCH: List of keywords used to determine if a metadata record is geospatial. If more than one keyword is needed, separate each keyword with a vertical bar "|"

  • LIST_SEARCH_ATTRIBITE: Name of the attribute in plain text (ex.: tag{}.display_name) used to search for a string match from the KEYWORD_SEARCH parameter
