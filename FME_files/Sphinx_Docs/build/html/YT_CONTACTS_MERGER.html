YT_CONTACTS_MERGER

Documentation du Custom Transformer FME

Description

This transformer uses thoses attributes:

  • author_email
  • contact
  • contact_role
  • default_contact_email
  • default_contact_role

It adjust the format of thoses attributes, set default values if needed for author_email and contact_role and output a list contact{}.role and contact{}.email.
