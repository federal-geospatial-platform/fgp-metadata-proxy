DEFAULT_ATTRIBUTE_MAPPER_NG

Documentation du Custom Transformer FME

Description

This custom transformer maps each value pair (key, value) from the lookup table as attribute into each metadata record. If there are 25 value pairs in the lookup table each metadata records will have 25 new attributes.

Input Ports

  • DATA_INPUT: The input metadata records to process.

Output Ports

  • OUTPUT: The process output metdata records with new attributes (key, value).

Parameters

  • FEATURE_TYPE: Name of the CSV file to read (without .csv extension);

  • IN_CSV_LOOKUP_TABLES_DIR: Name of the directory path containing the CSV file to read;

  • P-T_ABBR: Two letters abbreviation of the province or the territory.
