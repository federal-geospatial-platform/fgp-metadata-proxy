NT_CATALOGUE_READER

Documentation du Custom Transformer FME

Description

This Custom Transformer is used to read the metadata of the North West Territories open data portal (https://www.geomatics.gov.nt.ca/en/resources). The portal is not offering a public API to access the metdata information so, this custom transformer is reading all the XHTML code and flatten it in order to parse it and extract the needed metadata information.

Note: There is no option to read only one record with a specific identifier.

Input Ports

  • Input: One feature to start the extraction of all the records from the North West Territories open data portal.

Output Ports

  • DATA: All the metadata records from the DATA catalog
  • MAPS: All the metadata records from the MAPS catalog
  • WMS: All the metadata records from the WMS catalog
  • REPORTS: All the metadata records from the REPORTS catalog
  • DOCUMENTS: All the metadata records from the DOCUMENTS catalog
  • MAPS_VIEWERS: All the metadata records from the MAPS_VIWERS catalog

Parameters

  • DATA_METADATA: Flag to enable or disable reading the DATA catalog (default: Yes).
  • MAPS_METADATA: Flag to enable or disable reading the MAPS catalog (default: Yes).
  • WMS_METADATA: Flag to enable or disable reading the WMS catalog (default: Yes).
  • REPORTS_METADATA: Flag to enable or disable reading the REPORTS catalog (default: Yes).
  • DOCUMENTS_METADATA: Flag to enable or disable reading the DOCUMENTS catalog (default: Yes).
  • MAPS_VIEWERS_METADATA: Flag to enable or disable reading the MAPS_VIEWERS catalog (default: Yes).
